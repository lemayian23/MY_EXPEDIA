# MY_EXPEDIA - Flight Booking System

A fully functional flight booking web application inspired by Expedia, built with PHP, MySQL, and Bootstrap. Developed as a Software Engineering Laboratory project at the Technical University of Kenya and enhanced with complete CRUD functionality, stored procedures, and a modern UI.

## Features

- ✈️ **Flight Search** – Search flights by origin, destination, and date
- 💺 **Booking Management** – Passenger registration and seat selection
- 🌍 **Country & City Management** – Add/edit/delete countries and cities
- 🏢 **Airline Management** – Manage airline details and logos
- 📊 **Database Stored Procedures** – All core operations use MySQL procedures
- 📱 **Responsive UI** – Bootstrap 5 frontend works on desktop and mobile
- 🔍 **Real-time Availability** – Shows available seats per flight

## Technology Stack

| Component | Technology |
|-----------|------------|
| Backend | PHP 8.0+ (PDO) |
| Database | MySQL / MariaDB |
| Frontend | HTML5, CSS3, JavaScript, Bootstrap 5 |
| Server | Apache (XAMPP / WAMP) |
| AJAX | Fetch API for asynchronous search & booking |
